# Changelog

All notable changes to this plugin should be recorded in this file.

The format is based on Keep a Changelog, and versions follow Semantic Versioning.

## [0.2.0] - 2026-03-26

### Changed

- Renamed plugin from Console to Cathode across all identifiers, CSS classes, and documentation.
- Terminal font now inherits from Obsidian's monospace font setting and updates live on theme changes.

### Added

- Shift+Enter sends a newline instead of submitting in CLI apps like Claude Code.
- Added CLAUDE.md with project context and versioning conventions.
- Added versions.json for Obsidian version compatibility tracking.

## [0.1.1] - 2026-03-11

### Changed

- Adopted a Semantic Versioning workflow for ongoing development and release tracking.
- Added agent instructions requiring future changes to bump the plugin version and update this changelog in the same task.
